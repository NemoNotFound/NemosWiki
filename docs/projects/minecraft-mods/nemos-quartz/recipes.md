# Recipes

## Quartz Brick Blocks, Quartz Pressure Plate & Quartz Button

You can craft these items using the standard crafting recipes in the crafting table.    
Additionally, all of them can also be crafted using the **stonecutter**.

## Cracked Quartz Brick Blocks

Cracked variants are created by **smelting** their quartz brick counterparts in a furnace:

- **Quartz Bricks** -> Cracked Quartz Bricks
- **Quartz Brick Stairs** -> Cracked Quartz Brick Stairs
- **Quartz Brick Slab** -> Cracked Quartz Brick Slab
- **Quartz Brick Wall** -> Cracked Quartz Brick Wall

## Quartz Pot

To craft a **Quartz Pot**, place **three Quartz** in the crafting table in the following pattern:

`_` = empty slot    
`#` = quartz

`#_#`    
`_#_`
