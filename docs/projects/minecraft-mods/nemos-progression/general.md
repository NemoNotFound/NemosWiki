# Nemo's Progression

Nemo's Progression introduces a new progression system. 
Starting from copper tier, tools/weapons/armor can no longer be crafted trough regular recipes. 
Instead, upgrade templates are used to progress through equipment tiers, making every tier more valuable.

For more information about the upgrade templates, check [this](https://wiki.nemonotfound.com/projects/minecraft-mods/nemos-upgrade-templates/general).
